# Converter 🔁  
**Bit Angle Converter**  
A lightweight C++ project that converts bit-encoded values into meaningful angle representations in **degrees** and **radians**.

---

```Converter/

│    └── main.cpp
├── tests/

├── CMakeLists.txt
└── .gitignore```
